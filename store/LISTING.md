# Store listing draft — Cats & Rats

## Short description (132 chars max for Chrome)

Find and download images from any webpage. Simple grid, select, save. Built for team Cats & Rats.

## Detailed description

Cats & Rats helps you grab images from the page you’re looking at.

• Scan the active tab for images  
• Preview in a clean two-column grid  
• Select one or many, then download  
• Filter by minimum size  
• Works as a toolbar popup (Chrome) and sidebar-friendly panel (Firefox / Zen)

**Privacy:** Image lists stay on your machine. Downloads go through the browser. No account required. No analytics in v1.0.0.

**Permissions**

• **activeTab / scripting** — read image URLs from the page you choose  
• **downloads** — save selected files  
• **storage** — reserved for preferences  
• **Host access (http/https)** — resolve and fetch image URLs on sites you visit  

## Single purpose

Discover and download images from the current web page.
